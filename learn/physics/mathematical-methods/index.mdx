# Mathematical Methods

Mathematical methods for physics provides the essential mathematical tools and techniques needed to solve physical problems.

## Topics

- Complex Analysis
- Vector Calculus
- Variational Calculus
- Linear Algebra
- Tensor Analysis
- Group Theory
- Differential Equations
- Integral Transforms
- Fourier Analysis
- Laplace Transforms
- Special Functions
- Bessel Functions
- Legendre Polynomials
- Spherical Harmonics
- Green's Functions
- Probability Theory
- Statistical Methods
- Stochastic Processes
- Numerical Methods
- Perturbation Theory
- Asymptotic Analysis
- Differential Geometry
- Lie Groups
- Matrix Theory
- Distribution Theory
- Functional Analysis
- Path Integrals
- Complex Variables
- Conformal Mapping
- Contour Integration

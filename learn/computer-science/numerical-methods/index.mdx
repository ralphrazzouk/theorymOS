# Numerical Methods

Numerical methods focus on developing and analyzing algorithms for solving mathematical problems using numerical approximation.

## Topics

- Tensor Approximation
- Spectral Norm Error
- Preconditioned Length-Squared Sampling
- Subspace Embeddings
- Randomized Matrix Multiplication

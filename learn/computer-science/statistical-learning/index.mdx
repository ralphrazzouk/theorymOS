# Statistical Learning

Statistical learning combines statistical methods with machine learning techniques to extract insights from data.

## Topics

- Binary Classification
- Regression Analysis
- Best Linear Unbiased Prediction
- Gauss Markov Theorem
- Principal Component Analysis
- Naive Bayes
- Logistic Regression
- Gaussian Mixture Models
- Linear Models
- Wavelet Smoothing
- Bias-Variance Tradeoff
- Multidimensional Splines
- Kernel Density Estimation
- Cross Validation
- Bootstrapping
- Inference Models
- Random Forests
- Latent Variable Modelling
- Boltzmann Machines

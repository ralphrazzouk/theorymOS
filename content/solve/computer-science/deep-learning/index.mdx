# Deep Learning

Deep learning focuses on training artificial neural networks to learn from data and make predictions or decisions.

## Topics

- Large Language Models
- Transformers
- Neural Networks
- Reinforcement Learning
- Bayesian Networks
- Recurrent Neural Networks
- Convolutional Neural Networks
- Gradient Descent
- Policy Approximation Methods
- Long-Short Term Memory
- Variational Auto Encoders
- Generative Adversarial Networks
- Stable Diffusion

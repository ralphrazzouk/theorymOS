# Learning Theory

Learning theory provides mathematical frameworks for understanding how and why machine learning algorithms work.

## Topics

- Perceptron
- Support Vector Machines
- Weighted Majority
- PAC Learning
- VC Dimension
- Ada Boost
- Random Graphs
- Fixed Point Iterations
- Empirical Risk Minimization
- Uniform Convergence
- Multi-armed Bandit
- Regret Minimization
- K-junta Problem

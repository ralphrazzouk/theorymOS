# Computational Statistics

Computational statistics focuses on the implementation of statistical methods using computational techniques and algorithms.

## Topics

- Monte Carlo Methods
- Sampling Methods
- Las Vegas Algorithms
- Metropolis Hastings
- Gibbs Sampling
- Geometric Random Walks

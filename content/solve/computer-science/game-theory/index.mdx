# Game Theory

Game theory analyzes strategic interactions between rational decision-makers in various contexts.

## Topics

- Zero-Sum Games
- Discrete-time Games
- Combinatorial Games
- Quantum Games
- Voting Theory
- Hierarchical Games
- von Neumann Minimax Theory
- Stochastic Games
- Nash Equilibrium
- Bayesian Games
- Bayesian Equilibrium
- Beliefs and Knowledge Models
- Monty Hall Problem
- Fixed-Point Theorems
- Auction Theory

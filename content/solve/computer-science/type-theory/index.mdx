# Type Theory

Type theory provides a formal framework for reasoning about types and their relationships in programming languages.

## Topics

- Types
- Terms
- Judgements
- Rules of Inference
- Type Inhabitation
- Curry-Howard Correspondence
- Homotopy Type Theory
- Atomic Terms
- Function Terms
- Lambda Terms
- Function Application
- Reductions

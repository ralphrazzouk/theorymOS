# Data Structures

Data structures are fundamental building blocks in computer science that enable efficient data organization and manipulation.

## Topics

- Stack
- Queue
- Hashmap
- Trie
- Array
- Linked List
- Tree
- Self-Balancing Trees
- Graphs
- Augmented Data Structures
